# 指针仪表检测

本项目通过计算机视觉技术来实现对固定位置和尺寸的指针仪表的检测。项目将图像处理和分析应用于摄像头捕获的图像，识别指针位置和角度，进而计算相应的测量值（MPa）。本项目使用OpenCV库来进行图像处理和分析。

### 文件结构

```shell
.
├── 1.jpg  # 测试图片
├── 2.jpg
├── 3.jpg
├── 4.jpg
├── 5.jpg
├── README.md
└── main.py  # 主程序
```

## 项目要求

- 摄像头的位置固定。
- 摄像头捕获的图像尺寸为固定大小（320 * 240）。

## 运行指南

修改 `main.py` 中第 7 行 `file_name` 变量，将其设置为待处理图像的文件名。

### 运行测试

```shell
python main.py
```
